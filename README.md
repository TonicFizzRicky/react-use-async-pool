# react-use-async-pool

`react-use-async-pool` 是一个 react hook,支持控制异步函数的并发控制

## 如何使用

### 安装依赖

```bash
pnpm i react-use-async-pool

```

### 使用

通过 ESM 方式导入

```ts
import {useAsyncPool} from 'react-use-async-pool'

```

## 开发

你可以 Fork 此仓库进行开发

Fork 后可以安装依赖

```bash
pnpm i

```

启动本地开发环境

```bash
pnpm dev
```

打包文档

```bash
pnpm build-docs
```

打包为 ssr 静态站点

```bash
pnpm ssr-docs
```

打包库
```bash
pnpm build-lib
```
